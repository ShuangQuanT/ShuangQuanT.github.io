##两个#文章

###三个#是三级标题

-列表1(-名字)

​ a子列表(a)

###链接:

““前面是名字,后面是链接

###插入图片

​ 和链接一样后面是图片链接

###字体加粗* *可以多个

###代码块单行用esc下面的键的 两个瓢`

###多行还是多个至少三个(一对)```

最好还是下个typora

  • 主题设置:

    • 下载相关主题
    • 然后去theme下配置相关文件
    • 然后配置_config.yml文件
    • 最后是生成tags等标签一般有说明

  • How to write:

    • hexo new或者n name会自动生成到文件source/_posts文件里面
    • 或者直接去文件夹里面生成,最好还是命令
    • 然后编辑文件tags: 标签名字
    • categories: 分类名字
    • ——线后面正文

hexo generate //静态文件
hexo server //浏览静态文件
hexo deploy //同步到博客

密码为动态API
